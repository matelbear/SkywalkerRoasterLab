LED Status Indicator Guide

🟢 Solid Green: Standby mode, Bluetooth connected.

🟢 Blinking Green: Standby mode, Bluetooth disconnected / waiting for connection.

🟠 Solid Orange: Heating/Roasting in progress (Heat > 0).

🔵 Solid Blue: Cooling mode active (Cooling).

🟡 Solid Yellow: Only Fan or Drum is running (no active heating).

🔴 Fast Blinking Red: High-temperature Emergency Stop (E-Stop) triggered.
